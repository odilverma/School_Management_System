# School_Management_System
A Python-based School Management System using OOP concepts, with data stored in a text file for managing students and teachers.


## Description
A Python-based School Management System built using Object-Oriented Programming (OOP).  
It allows managing **students and teachers** with features like add, update, delete, search, and view records.  
All data is stored in a text file (`school.txt`) for persistence.


## Features
- Add new students with name, age, and marks
- Add new teachers with name, age, salary, and subject
- Display all records (students and teachers)
- Search student by name
- Update existing records (age, marks, salary, subject)
- Delete records
- Data persistence using `school.txt`


## Project Files
- `school_system_project.py` → Main Python code file  
- `school.txt` → Data file (stores student and teacher records)  
- `README.md` → Documentation  


## Tech Stack
- **Language:** Python  
- **Concepts:** Object-Oriented Programming (Inheritance, Polymorphism)  
- **Storage:** Text file (`school.txt`)
